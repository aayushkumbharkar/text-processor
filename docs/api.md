# API Reference

## `count_words(file_path: str) -> int`
Counts the total number of words in the file.

## `count_chars(file_path: str) -> int`
Counts the total number of characters in the file.

## `count_lines(file_path: str) -> int`
Counts the total number of lines in the file.

## `find_word(file_path: str, word: str) -> int`
Finds the occurrences of a word in the file.

## `replace_word(file_path: str, old_word: str, new_word: str) -> str`
Replaces a word in the file and saves a new version.
